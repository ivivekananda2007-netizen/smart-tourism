## 🛠️ YOUR SUPPORT GUIDE

### **If Friend Gets Error on Clone:**
- Check they copied link correctly
- Check they have Git installed
- Try running: `git --version`
- If not installed, download from git-scm.com

### **If Friend Can't Install Node.js:**
- Send them: nodejs.org
- Make sure they download LTS version
- Wait for full installation

### **If Friend Can't Create .env:**
- Show them the .env.example file
- Copy values to new .env
- Replace placeholder values

### **If Friend Can't Connect MongoDB:**
- Check MongoDB URI is correct
- Check IP address is whitelisted
- Check username & password are right
- Check database name matches

### **If Friend Can't Get API Key:**
- Go to openweathermap.org together
- Sign up for free account
- Copy API key
- Put in .env file

### **If Server Won't Start:**
- Check all npm packages installed
- Check .env file exists
- Check MongoDB is connected
- Check ports 5000 & 5173 are free
- Try restarting
